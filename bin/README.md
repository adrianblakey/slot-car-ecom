You'll find the [latest firmware in the github release repo for ESCape32.](https://github.com/neoxic/ESCape32/releases)

The 2 binaries here are an experiment that are built from a fork https://github.com/adrianblakey/ESCape32 of the ESCape32 repo. They permit freq_min and freq_max to be set up to 72kHz and 144kHz respectively this might help to start high Kv motors.
